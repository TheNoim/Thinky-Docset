# Thinky.IO Docset for Dash

## Generate:

1. Install Node.JS
2. Install dependencies
```bash
yarn # If you are using yarn
# OR
npm install # If you never heard of yarn
```
3. Build Docset
```bash
yarn run build && node Index.js
# Or
npm run build && node Index.js
```
4. Now you can use the docset :)
